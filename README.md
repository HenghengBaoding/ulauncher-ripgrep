# ulauncher-ripgrep
> Find file content by riggrep in ulauncher

- First,You should install ripgrep and ulauncher.
- Then clone the project in your extension directory,like ~/.local/share/ulauncher/extensions.
- OK,reboot ulauncher,you can use it by input rg.
  
> [!NOTE]
> - To ensure search performance, hidden directories are not included in the matching process. If needed, you can set the search_path to hidden directories in the extension preferences. Multiple directories are supported, separated by spaces.
> - This extension does not support regular expression matching; it only performs exact matches.
